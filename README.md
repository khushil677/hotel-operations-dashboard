# Hotel Operations Dashboard

## Business Impact
This dashboard helps hotel management:
- Monitor performance in real time
- Identify revenue growth opportunities
- Optimize pricing and occupancy strategies
- Make data-driven operational decisions

## Overview
This project focuses on analyzing hotel performance data to generate actionable insights for business decision-making. The dashboard provides a clear view of key hospitality metrics such as revenue, occupancy rate, and Average Daily Rate (ADR), helping management evaluate performance trends and optimize operations.

## Objective
The goal of this project was to simulate a real-world hospitality analytics scenario by transforming raw hotel data into an interactive dashboard that supports strategic decisions in revenue management and operational efficiency.

## Tools & Technologies
- Power BI
- Microsoft Excel
- MySQL (for data preparation)
- DAX (Data Analysis Expressions)

## Key Features
- Executive KPI cards displaying Total Revenue, Occupancy Rate, and ADR
- Year-over-Year (YoY) growth analysis for performance comparison
- Time-based trend analysis (monthly and yearly)
- Revenue distribution across different hotel segments
- Interactive filters for dynamic data exploration

## Data Preparation
- Cleaned and structured raw data using Excel
- Removed inconsistencies and handled missing values
- Modeled relationships between datasets in Power BI
- Created calculated measures using DAX for KPIs and growth metrics

## Key Insights
- Identified peak revenue periods and seasonal demand trends
- Observed fluctuations in occupancy impacting revenue performance
- Highlighted areas where pricing strategy could be optimized
- Provided insights to support better operational and financial planning
